## Completed Courses block

The Completed Courses block displays the current user's completed Moodle courses in a compact dashboard block.

## Features

*   Displays completed courses for the current user.
*   Provides block instance settings for course name display, sort order, row limit, category filtering, hidden courses, grades, categories, date format, and link target.
*   Shows paginated completed course results.
*   Exports the visible completed course list as CSV.

## Installation

Install this repository as `blocks/completedcourse`, then run the standard Moodle upgrade process.

No Composer, npm, or build step is required.
